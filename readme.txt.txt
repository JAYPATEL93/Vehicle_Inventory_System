c - create new car with interactive gui

l - list all cars by reading from the files.

1...5 - will load numbered car

p - command will update the price of the last car in memory, the last car in memory will
    be determined by the operation c (create new car) or command [1...5].

s - save last two cars in memory after modification.

q - quit the program
